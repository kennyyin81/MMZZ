# points-wine-miniapp

微信云开发小程序，当前主要能力：

- 积分申请、喝酒申请与审批
- 审批人快捷入口直接调分
- 站内消息通知
- 酒款浏览、收藏、评分、留言
- 酒款相似推荐（算法自动计算，定时 + 手动更新）
- 首页内嵌"微醺日历"与记录维护
- 待办工作与加分工作

## 启动前准备

1. 复制 `miniprogram/config/cloud.example.js` 为 `miniprogram/config/cloud.js`
2. 在 `miniprogram/config/cloud.js` 填写真实云环境 `env`
3. 部署云函数 `cloudfunctions/api` 和 `cloudfunctions/wine-scheduler`
4. 按 [docs/database.md](d:/vscode/Projects/points-wine-miniapp/docs/database.md) 创建集合和索引
5. 重新编译小程序

说明：

- 真实云环境配置仅保存在本地 `miniprogram/config/cloud.js`
- 仓库不会提交真实云环境配置

## 建议验证

1. 首页点击"当前积分"可进入积分流水
2. 首页点击"待我审批"可进入待审批列表
3. 首页点击"未读消息"可进入消息中心
4. 审批人首页可进入"直接调分"
5. 酒详情收藏后，"我的 > 我的收藏"能看到对应酒款
6. 加分工作在未绑定审批人时无法开启
