# points-wine-miniapp

微信云开发小程序，包含以下功能：

- 积分申请与喝酒申请
- 一对一审批关系
- 审批人快捷入口直接加分 / 减分
- 站内消息通知
- 酒款浏览、评分、留言
- 首页内嵌「日历记酒」与喝酒记录维护
- 品酒师维护酒款内容
- 待办工作（普通待办 / 加分待办）

## 启动前准备

1. 复制 `miniprogram/config/cloud.example.js` 为 `miniprogram/config/cloud.js`，并填写真实云环境 `env`
2. 部署云函数 `cloudfunctions/api`
3. 按 `docs/database.md` 创建集合和索引
4. 重新编译小程序

说明：

- 真实云环境配置保存在本地文件 `miniprogram/config/cloud.js`
- 仓库仅提交示例文件 `miniprogram/config/cloud.example.js`

## 关键页面

- 首页：个人信息、积分、待我审批、快捷入口、日历记酒
- 直接调分：审批人对当前被审批对象直接加分或减分
- 酒：酒款内容浏览、收藏、评分、留言、相似推荐跳转
- 记录：工作备忘录、待办管理
- 我的：修改资料、审批关系、邀请审批人、我的收藏
- 审批中心：待审批与历史审批
- 酒款维护：品酒师后台
- 喝酒记录详情：新增/编辑/删除单条记录

## 部署后建议验证

1. 用户提交加分申请，审批通过后余额增加
2. 用户提交喝酒申请，审批通过后按 `cost_points` 扣减积分
3. 审批人查看待审批列表，申请人显示昵称
4. 首页点击“当前积分”进入积分流水
5. 首页点击“待我审批”进入待审批列表
6. 首页点击“未读消息”进入消息中心
7. 审批人在首页快捷入口点击“直接调分”可为当前被审批对象直接调分
8. 当被审批对象积分为 0 或负数时，审批人仍可继续直接减分或加分
9. 酒类详情页点击收藏图标后，“我的 > 我的收藏”可看到对应酒款
10. 酒类详情页“相似推荐”可按风味相近顺序跳转到其他酒款详情
