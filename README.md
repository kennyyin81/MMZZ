# points-wine-miniapp

微信云开发小程序，当前主要能力：

- 积分申请、喝酒申请与审批
- 审批人快捷入口直接调分
- 站内消息通知
- 酒款浏览、收藏、评分、留言
- 首页内嵌“喝酒日历”与记录维护
- 待办工作与加分工作

## 启动前准备

1. 复制 `miniprogram/config/cloud.example.js` 为 `miniprogram/config/cloud.js`
2. 在 `miniprogram/config/cloud.js` 填写真实云环境 `env`
3. 部署云函数 `cloudfunctions/api`
4. 按 [docs/database.md](d:/vscode/Projects/points-wine-miniapp/docs/database.md) 创建集合和索引
5. 重新编译小程序

说明：

- 真实云环境配置仅保存在本地 `miniprogram/config/cloud.js`
- 仓库不会提交真实云环境配置

## 关键页面

- 首页：个人信息、积分、待我审批、快捷入口、喝酒日历
- 直接调分：审批人给当前被审批对象直接加分 / 减分
- 酒：酒款浏览、收藏、评分、留言、相似推荐
- 记录：待办工作列表与详情
- 我的：修改资料、审批关系、邀请审批人、我的收藏
- 酒款维护：品酒师维护酒款内容

## 酒款图片建议

- 建议比例：`1:1` 或 `4:5`
- 建议尺寸：不低于 `800 × 800`
- 更稳妥的上传尺寸：`1000 × 1000` 或 `1200 × 1500`
- 主体尽量居中，避免在预览中被裁切

## 建议验证

1. 首页点击“当前积分”可进入积分流水
2. 首页点击“待我审批”可进入待审批列表
3. 首页点击“未读消息”可进入消息中心
4. 审批人首页可进入“直接调分”
5. 酒详情收藏后，“我的 > 我的收藏”能看到对应酒款
6. 加分工作在未绑定审批人时无法开启
