# 酩酩之中 PRD

## 产品目标

- 把积分、审批、消息提醒放到同一个小程序里完成
- 把酒款内容浏览、互动和个人记录整合到同一套体验里
- 酒友广场让用户分享喝酒记录，形成公开内容流

## 角色

- `USER`：提交申请、查看积分、浏览酒款、维护个人记录
- `APPROVER`：审批绑定用户的申请与加分工作
- `ADMIN`：保留系统管理权限标识；当前无前台角色管理功能
- `SOMMELIER`：维护酒款内容和酒馆内容

## 核心规则

### 积分

- 加分申请审批通过后增加积分
- 喝酒申请审批通过后扣减积分
- 审批人可通过首页快捷入口直接调分
- 直接调分写入 `points_ledger`，来源为 `manual_adjust`
- 直接调分允许当前余额和调整后余额为负数
- 当用户当前积分为负数时，加分申请和加分工作审批通过仍可正常入账

### 待办工作

- 待办分为普通工作和加分工作
- 普通工作完成后直接标记为完成
- 加分工作完成后提交审批，审批通过后入账积分
- 未绑定审批人时，前端不允许开启加分工作
- 记录页列表只展示标题与标签，不展示状态文字与时间

### 审批关系

- 普通用户与审批人是一对一关系
- 用户可邀请另一位用户成为自己的审批人
- 若当前关系下仍有待审批申请，双方不能直接解绑

### 酒款模块

- 酒款由 `wine_topic` 集合维护
- 用户可收藏、评分、留言
- 同一用户对同一酒款仅保留 1 条评价记录
- 酒详情展示封面、名称、类别 · 度数、介绍、风味、口感简图、基酒与原料、口感解读、背景故事、相似推荐
- 相似推荐由算法自动计算，每周一凌晨 3 点自动更新，也支持在酒款维护页手动触发
- 相似推荐算法从风味标签、类别、基酒、原料、口感五个维度加权评分，每款酒取 Top-3
- 酒款维护当前字段包含：名称、类别、酒精度、风味、口感等级、基酒、原料、口感解读、背景故事、相似推荐、封面
- 酒百科浏览页支持回到顶部浮动按钮

### 酒馆模块

- 底部“酒馆”Tab 支持在酒馆和酒款之间切换浏览
- 酒馆列表支持按省份、城市和关键词筛选
- 酒馆预览卡片不额外标注“酒馆/酒款”类型
- 酒馆详情展示封面、名称、区域、类型、人均、评分、推荐亮点、营业信息、地址、电话和环境相册
- 用户可按与酒款一致的星级样式给酒馆评分
- 同一用户对同一酒馆仅保留 1 条评分记录
- 酒馆维护由 `SOMMELIER` 角色通过后台入口完成
- 酒馆维护支持地图搜索填入酒馆名、地址、经纬度，并尽量解析省份、城市和区域
- 酒馆删除采用下架策略，不物理删除历史数据

### 酒友广场

- 酒友广场不是独立发帖系统，是从喝酒记录分享出来的公开内容流
- 用户在添加记录时可打开"保存后分享到酒友广场"开关
- 已保存记录可选择"分享到酒友广场"进入发布预览页，点击分享时先自动保存修改再跳转
- 发布预览页带入原记录内容，允许选择封面图、设置地点公开范围、控制"其他"笔记是否展示
- 地点公开范围：显示店名（含完整地址）/ 只显示城市或区域 / 不显示地点
- 同一条喝酒记录不能重复发布广场动态
- 删除广场动态不删除原喝酒记录
- 广场预览卡片不展示价格，动态详情页展示价格
- 广场页面以信息流卡片展示，包含头像、昵称、时间、地点、封面、酒名、口感、环境、点赞/评论/收藏数
- "我的收藏"页面支持 Tab 切换查看收藏酒款和收藏的广场动态
- "我的"页面"酒藏与动态"模块包含"我的收藏"和"我的动态"两个入口
- "我的动态"页面展示当前用户发布到广场的所有动态

## 首页规则

- "当前积分"点击后进入积分流水
- "待我审批"点击后进入待审批列表
- "未读消息"点击后进入消息中心
- "未读消息"展示在昵称同一行右侧
- "未读消息"计数不重复包含待审批提醒
- "当前积分""待我审批"采用"标题：数值"的紧凑形式
- 审批人存在被审批对象时，快捷入口显示"直接调分"
- 首页展示内嵌微醺日历

## 页面结构

- 首页
- 酒馆
- 广场
- 我的

主要页面：

- 首页：个人信息、积分、待我审批、快捷入口（含"我的记录"）、微醺日历
- 直接调分：审批人查看当前被审批对象并直接调分
- 酒列表：点击图片进入详情
- 酒详情：收藏、评分、留言、相似推荐
- 酒友广场：信息流展示分享的喝酒记录，支持点赞、评论、收藏
- 广场动态详情：完整内容、评论列表、互动操作
- 发布预览页：选择封面、设置地点公开范围、控制"其他"笔记是否展示
- 我的收藏：Tab 切换查看收藏酒款和收藏的广场动态
- 我的动态：查看当前用户发布到广场的所有动态
- 记录：待办列表、日期筛选、详情
- 我的：修改资料、审批关系、邀请审批人、酒藏与动态（我的收藏、我的动态）
- 酒款维护：品酒师后台，现有酒款通过名称搜索，支持手动更新相似推荐
- 酒馆维护：品酒师后台，支持新增/编辑/下架酒馆和地图搜索填入
- 消息中心：支持一键已读、勾选、全选、批量删除，并可跳转相关业务页面
- 审批历史：支持勾选、全选、批量删除
